# Contributing to MTProto Proxy Hub

Спасибо за интерес к проекту! Мы рады любому вкладу.

## 🚀 Как начать

1. Fork репозитория
2. Создайте ветку для вашей фичи (`git checkout -b feature/amazing-feature`)
3. Commit изменения (`git commit -m 'Add amazing feature'`)
4. Push в ветку (`git push origin feature/amazing-feature`)
5. Откройте Pull Request

## 📝 Стандарты кода

### Frontend (TypeScript/React)
- Используйте TypeScript для всех новых файлов
- Следуйте существующему стилю кода
- Используйте функциональные компоненты с хуками
- Добавляйте типы для всех props и state

### Backend (Python)
- Следуйте PEP 8
- Используйте type hints
- Документируйте функции с помощью docstrings
- Используйте async/await для асинхронных операций

### CSS
- Используйте существующие CSS переменные
- Следуйте naming convention (kebab-case)
- Добавляйте стили для темной темы

## 🎨 Дизайн

- Основной цвет: Ocean Teal (#14b8a6)
- Темная тема: Slate Night (#334155, #475569, #1e293b)
- Шрифты: SF Pro Display (UI), SF Mono (code)
- Используйте Material UI иконки

## 🐛 Баг репорты

При создании issue укажите:
- Описание проблемы
- Шаги для воспроизведения
- Ожидаемое поведение
- Скриншоты (если применимо)
- Версия браузера/ОС

## 💡 Предложения фич

Мы открыты к новым идеям! Создайте issue с тегом `enhancement` и опишите:
- Что вы хотите добавить
- Зачем это нужно
- Как это должно работать

## ✅ Checklist перед PR

- [ ] Код работает локально
- [ ] Нет ошибок в консоли
- [ ] Код следует стандартам проекта
- [ ] Добавлены комментарии для сложной логики
- [ ] README обновлен (если нужно)
- [ ] Темная тема работает корректно

Спасибо за ваш вклад! 🎉
