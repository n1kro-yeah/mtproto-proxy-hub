# Changelog

Все значимые изменения в проекте будут документированы в этом файле.

## [1.0.0] - 2024-XX-XX

### ✨ Добавлено
- Автоматическая загрузка прокси из GitHub при старте
- Темная тема (BETA) с Slate Night палитрой
- Сортировка прокси по задержке, статусу, типу, стране
- Плавные анимации с Framer Motion
- Пульсирующие индикаторы для всех статусов
- Material UI иконки вместо эмодзи
- SF Pro Display и SF Mono шрифты
- Бейдж BETA на кнопке темной темы
- Компактный дизайн карточек
- Бирюзовая цветовая схема (Ocean Teal)

### 🎨 Изменено
- Обновлен дизайн с Soft UI на Ocean Teal
- Кнопки теперь белые с бирюзовой обводкой
- Улучшена анимация сортировки
- Оптимизирована загрузка шрифтов

### 🐛 Исправлено
- Исправлено мерцание карточек при сортировке
- Исправлена высота карточек в grid
- Исправлены стили для темной темы

### 🔧 Техническое
- Добавлен .gitignore
- Добавлен LICENSE (MIT)
- Обновлен README с новыми фичами
- Добавлены шаблоны для issues
- Добавлен CONTRIBUTING.md
- Добавлен DEPLOYMENT.md
